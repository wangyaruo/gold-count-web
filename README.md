# 黄金小账本

一个用于记录黄金买入、卖出、克数、单价和费用的本地网页小账本。它会根据手动输入的当前金价计算持仓估值、持仓浮盈、已实现盈亏和总盈亏。

## 功能

- 记录买入和卖出交易。
- 支持日期、克数、单价、费用和备注。
- 手动保存当前金价。
- 自动计算持仓克数、剩余成本、当前估值、持仓浮盈、已实现盈亏和总盈亏。
- 支持交易编辑、删除和买入/卖出筛选。
- 数据保存在当前浏览器的 `localStorage`，刷新页面后仍然保留。

## 技术栈

- TypeScript
- Vite
- Element Plus


## 盈亏计算口径

- 买入成本 = 克数 × 买入单价 + 费用。
- 卖出收入 = 克数 × 卖出单价 - 费用。
- 已实现盈亏按 FIFO 计算：最早买入的黄金先被卖出。
- 剩余成本是卖出匹配后仍持有黄金的成本。
- 当前估值 = 当前金价 × 当前持仓克数。
- 持仓浮盈 = 当前估值 - 剩余成本。
- 总盈亏 = 已实现盈亏 + 持仓浮盈。

## 数据说明

当前版本没有后端和登录，所有数据只保存在使用该网页的浏览器中。更换浏览器或设备不会自动同步数据。
